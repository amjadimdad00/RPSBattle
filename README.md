# RPSBattle

## About

### RPSBattle Game

RPSBattle is a modern take on the classic Rock-Paper-Scissors game, offering both single-player and multiplayer modes. In this game, players can test their strategic skills by choosing Rock, Paper, or Scissors to outmaneuver their opponent. - **HTML5**: For structuring the game interface.

### How to Play

1. Choose Rock, Paper, or Scissors.
2. Your choice will be compared against the opponent's choice.
3. Rock beats Scissors, Scissors beats Paper, and Paper beats Rock.
4. The first player to reach the set number of wins is the champion!
